gMusic privacy policy

To use gMusic player you need to provide your Google Account's email and password. If you do not have one, you need to create Google Account. Please make sure that you have subscribed to Google Music service. For more details go to the Google Music site.

gMusic can store your Google Account user name and password, you will be asked on the authentication page if you want to do this or not. Password will be stored in encrypted Windows Storage.

We do not collect any personal information. But we can collect application health information, which does not include any personal information, and this can be collected only by turning on special settings in Support tab.

Because we use Google Music API - you need to be acquainted with Google Account Privacy Policy.
